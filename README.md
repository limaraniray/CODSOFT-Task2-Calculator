## 📖 About the Project

This project is a **Calculator application** designed to perform basic arithmetic operations through a clean command-line interface. The user inputs two numbers and selects an operation — the result is calculated and displayed instantly. It handles edge cases like division by zero and non-numeric input with helpful error messages.

This is **Task 2** of the CodSoft Python Programming Internship.

---

## ✨ Features

| Feature           | Description                                  |
| ----------------- | -------------------------------------------- |
| ➕ Addition       | Add two numbers                              |
| ➖ Subtraction    | Subtract second from first                   |
| ✖️ Multiplication | Multiply two numbers                         |
| ➗ Division       | Divide with zero-division protection         |
| 📐 Modulus        | Get the remainder                            |
| 💪 Power          | Exponentiation (a \*\* b)                    |
| ❌ Validation     | Handles non-numeric and invalid input        |
| 🔄 Loop           | Perform multiple calculations in one session |

## ▶️ How to Run

### Prerequisites

- Python 3.x must be installed → [Download Python](https://www.python.org/downloads/)

### Steps

```bash
# Step 1: Clone the repository
git clone https://github.com/limaraniray/CODSOFT-Task2-Calculator.git

# Step 2: Navigate into the folder
cd CODSOFT-Task2-Calculator

# Step 3: Run the application
python task2_calculator.py
```

## 💻 Usage

```
========================================
         🧮 CALCULATOR APP
========================================
1. Addition       (+)
2. Subtraction    (-)
3. Multiplication (*)
4. Division       (/)
5. Modulus        (%)
6. Power          (**)
7. Exit
========================================
Choose an operation (1-7):
```

### ➕ Addition Example

```
Choose an operation (1-7): 1
Enter first number: 45
Enter second number: 30

✅ Result: 45 + 30 = 75
```

### ➗ Division by Zero

```
Choose an operation (1-7): 4
Enter first number: 10
Enter second number: 0

❌ Error: Division by zero is not allowed!
```

### 💪 Power Example

```
Choose an operation (1-7): 6
Enter first number: 2
Enter second number: 8

✅ Result: 2 ** 8 = 256
```
